# @megon/server

Core AI agent runtime for the Megon desktop application and CLI.

## Features

- Multi-provider AI support (Anthropic, OpenAI, Google, xAI, etc.)
- Tool execution (bash, file operations, web fetch, etc.)
- Session management with conversation history
- Plugin system for extensibility
- MCP (Model Context Protocol) support

## Development

```bash
bun install
bun run dev
```

## Testing

```bash
bun run test
```
